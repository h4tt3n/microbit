Logic

Making decisions in code

Computers use logic and selection to control what happens in different situations.

  • if, elif, else

    Use if to test if something is true:

    Code example:
    score = 1
    if score < 2:
    display.show(Image.SAD)

    This shows a sad face because the score is less than 2.

    Python uses indentations to tell the computer exactly what instructions should be carried out in each case, similar to the way it uses indentations to show which lines of code are in a loop.

  • Comparison operators

    You can compare numbers to make different things happen if they are equal, smaller, larger or not equal to something.

    Code example:
    number = 5
    if number == 5:
    display.scroll('Number is 5')

    Remember one equals sign is used to assign the value of a variable:

    number = 5

    but when testing the value, the ‘is equals to’ operator uses two equals signs:

    if number == 5:

  • Logical operators

    You can build more complex conditions using and, or and not.

    Use and to build conditions which are True if more than one thing is True. In this example, 'A+B' is shown only if button A and button B are pressed at the same time:

    Code example:
    while True:
    if button_a.is_pressed() and button_b.is_pressed():
    display.scroll('A+B')

  • Nested if

    You can put if statements inside each other. This is called ‘nesting’.

    Code example:
    score = 10
    if score > 5:
    display.scroll('Well done')
    if score > 9:
    display.scroll('Top marks!')

    This shows ‘Well done’ and then ‘Top marks!’ on the LED display.

  • Conditional Booleans

    Boolean variables can only have two values: True or False.

    You can use Boolean variables in if statements to write code that’s easy to read and understand.

    Code example:
    on = False
    while True:
    if button_a.was_pressed():
    on = not on
    if on:
    display.show(1)
    else:
    display.show(0)

    This example makes a toggle switch. It uses not to flip the value of the Boolean variable on every time you press button A.

    If its value was True, it becomes False. If its value was False, it becomes True.

  • Break using logic

    You can break out of a loop using conditional statements and break

    Code example:
    while True:
    if button_a.was_pressed():
    display.show('A')
    elif button_b.was_pressed():
    display.show('B')
    if accelerometer.was_gesture('shake'):
    break
    display.scroll('Game over')

    In this example, if you shake the micro:bit, the program breaks out of a while True loop and the words ‘Game over’ scroll on the display.

# Ultrasonic distance measurement with HC-SR04
# This version only works on Micro:Bit v2

# Wiring
# Micro:Bit - HC-SR04+:
# V0/GND - gnd
# 3V - vcc
# Pin0 - trig
# Pin1 - echo

from microbit import *
import utime
import machine

# Setup pins
trig = pin0
echo = pin1

# Ensure trig is low initially
trig.write_digital(0)

def get_distance():
# 1. Send a 10 microsecond pulse
trig.write_digital(1)
utime.sleep_us(10)
trig.write_digital(0)
# 2. Measure the duration of the Echo pulse in microseconds
# machine.time_pulse_us returns the time in us, or -1/-2 on timeout
duration = machine.time_pulse_us(echo, 1, 30000) # 30ms timeout
if duration > 0:
# 3. Calculate distance in cm
# Speed of sound: 0.0343 cm/us
distance = (duration * 0.0343) / 2
return distance
else:
return None

while True:
dist = get_distance()
if dist is not None:
print("Distance: {:.2f} cm".format(dist))
else:
print("Out of range")
sleep(100)
